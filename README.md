# 🚀 Campaign AI – Marketing Analytics Dashboard

---

## 📊 Overview

Campaign AI is a simulation-based marketing analytics dashboard that evaluates and compares advertising performance across multiple digital platforms.

It helps visualize campaign KPIs and provides automated insights for better marketing decisions.

---

## ✨ Features

- 📈 KPI Analytics (CTR, Engagement, Leads, Conversions, ROI)
- 🌐 Multi-platform comparison (TikTok, Instagram, Facebook, Snapchat, LinkedIn)
- 🎯 Campaign objective selection (Awareness, Traffic, Leads, Sales)
- 🧠 AI-style recommendation engine (automatic insights)
- 📊 Interactive charts for performance visualization
- 📁 Excel report export functionality

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js
- SheetJS (Excel export)

## 🎯 How It Works

1. Select campaign budget
2. Choose audience age group
3. Pick campaign objective
4. Select platforms
5. Run analysis
6. View AI insights + charts
7. Export Excel report
